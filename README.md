## Setup Instructions

1. Fork this repository by clicking on the "Fork" button at the top-right corner of this page. This creates a copy of the repository in your GitHub account.

2. Clone your forked repository

3. Then in your local project create a new branch within the name of the pattern that you want to add

```bash
git switch -c pattern-name
```
Perfect, you're ready to start coding!
